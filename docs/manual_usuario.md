# Manual de Usuario

## 1. Introducción
Esta aplicación permite realizar análisis transcriptómico
de manera interactiva mediante una interfaz web.

---

## 2. Flujo general de uso

1. Cargar un dataset (local o público)
2. Definir parámetros del análisis
3. Ejecutar análisis DESeq2
4. Explorar visualizaciones
5. Exportar resultados

---

## 3. Interpretación de resultados

- Los gráficos muestran genes diferencialmente expresados
- Los valores estadísticos deben interpretarse en contexto investigativo

---

## 4. Limitaciones

- No valida hipótesis clínicas
- Requiere conocimiento previo del dominio biológico
